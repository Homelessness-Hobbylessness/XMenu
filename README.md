# XMenu

Your multipurpose iOS Shortcut hub — packed with practical tools and quick actions to make everyday tasks faster and more fun.

---

## ✨ What is XMenu?
XMenu is an all‑in‑one iOS Shortcut built to streamline common tasks. It combines multiple utilities into a single, clean menu so you can access what you need in seconds.

---

## 🔧 Highlights
- **Multipurpose toolbox**: many tools in one place  
- **Fast access**: jump to tasks with minimal taps  
- **Easy to extend**: add or rearrange items as your workflow grows  
- **Clean experience**: simple, organized menu flow  

---

## 🚀 Get Started
1. Install the shortcut via our website:  
   https://xmenu.dev  
2. Open it in the Shortcuts app  
3. (Optional) Add to Home Screen for one‑tap access  

---

## 🧰 Example Use Cases
- Quick utilities and system actions  
- Productivity boosters  
- Handy everyday tools  
- Compact menu for your favorite automations
